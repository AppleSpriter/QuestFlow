# Changelog

All notable changes to QuestFlow will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/).

## v1.9 - 2026-06

任务推进体验与周期任务更新：围绕长期任务查找、推进记录、存档安全和每日/每周任务做增强。

### Added

- 新增单任务 Timeline 导出：同步与存档页可按任务导出制表符分隔的过程日志
- 新增当前专注 Todo List 拖动排序，支持按实际执行顺序调整子任务
- 新增存档覆盖/同步完成提示，明确展示本地/云端替换完成状态
- 新增 `/tags` 页面，可配置常用 Progress 标签并从预设颜色中选择样式
- 首页当前专注推进区支持选择 Progress 标签，日志保存标签快照，导出事件 Log 时包含标签列
- 任务标签新增“每日 / 每周”，用于周期性任务追踪，不影响 XP 加成
- 当前专注为每日/每周任务时新增“完成今日 / 完成本周”操作，完成后自动归档

### Changed

- 当前专注区域保留完整地图进度条，底部任务卡改为紧凑区域胶囊，避免后期地图过长溢出
- Active 任务卡仅在推进后移动到列表底部，单纯追踪/切换专注不改变位置；快速任务选取栏保持按创建时间固定排序，便于查找任务
- 历史任务卡支持直接编辑任务标签，可给已有任务补充每日/每周属性
- Progress 标签预设色扩展为更鲜明的多色方案，未选中标签弱化显示
- README、AGENTS 文档同步更新为 v1.9 / persist v15

### Fixed

- 修正 `--localstorage-file` 警告：服务端预渲染阶段改用 no-op storage provider，避免误访问浏览器 localStorage
- 修正任务焦点切换导致 Active 排序变化的问题，只有点击推进后才移动任务位置
- 修正技能检定 toast 偏右问题，现在居中显示
- 数据迁移至 v14：为历史存档补齐 Progress 标签集合与日志标签快照，并将 WebDAV 默认存档后缀升级为 v14
- 数据迁移至 v15：为历史存档补齐每日/每周任务标签和周期完成记录字段，并将默认存档后缀升级为 v15

## v1.8 - 2026-06

专长（Feat）与 Build 系统更新：在职业升级时加入永久选择，让职业等级从数值成长进一步演化为 Build 路线。

### Added

- 新增 36 个专长，覆盖学习流、专注流、幸运流、共鸣流、收集流、休息流
- 新增专长品质：普通、稀有、史诗、传奇
- 每个职业在 Lv4/Lv8/Lv12/Lv16... 获得专长点，并从随机 3 个专长中永久选择 1 个
- 新增 `/build` 页面，展示职业等级、主要流派、已选专长、职业倾向和当前 Build 评分
- 新增自动 Build 识别：命运编织者、博学者、执行大师、共鸣大师、收藏宗师、荒野旅者、Spellblade、奥术诡术师、Agent Architect
- 新增专长二次确认流程：先选卡片再确认，避免误触永久选择
- 新增稍后选择能力：可关闭专长选择弹窗，并在 `/build` 页面继续处理待选专长

### Changed

- Progress、技能检定、卷轴、共鸣、休息奖励接入专长加成
- Party Status 改为默认展示 4 个职业，支持展开/收起和 hover 查看职业详情，避免 Build/卷轴信息遮挡职业名与等级
- README、AGENTS 文档同步更新为 v1.8 / persist v13

### Fixed

- 数据迁移至 v13：为历史存档补齐 FeatState、待选择专长、短休/长休计数
- 修正专长/Build 展示与实际奖励不一致的问题：坚定意志、精力管理、共鸣核心、联动专家、双卷轴和社交达人现在与实际逻辑一致
- 修正新触发双卷轴共鸣会在同一次 Progress 被立即消耗的问题，现在只影响下一次卷轴奖励

## v1.7 - 2026-06

大版本更新：扩展为 **12 职业 + 66 个职业共鸣 + 共鸣圣殿图鉴**，让用户通过切换不同类型任务获得长期收集和成长目标。

### Added

- 新增 7 个职业：Paladin、Ranger、Druid、Warlock、Sorcerer、Monk、Barbarian
- 为 12 个职业补齐 60 条技能线、540 个 1~9 环技能名称
- 新增职业共鸣系统：连续推进两个不同职业任务触发双职业共鸣
- 新增 66 个职业共鸣组合，每个组合包含名称、独立徽章、奖励、描述
- 新增共鸣奖励池：+3 XP、+1 卷轴、-10 Fatigue、优势检定、幸运检定、双卷轴、长休卷轴
- 新增共鸣圣殿 `/resonance`：12×12 矩阵图鉴，展示已解锁组合、等级、触发次数和详情
- 新增共鸣等级：Lv1/Lv2/Lv3/Lv4/Lv5，对应 1/5/20/50/100 次触发
- 新增共鸣连锁：连续切换不同职业累计链数，达到 x5 额外奖励卷轴 ×1
- 新增首次发现共鸣弹窗和普通共鸣 1 秒轻量触发动画
- 新增 15 阶段任务地图，从起点营地推进到传奇之门
- 新增每个任务独立 Todo List，完成待办会自动推进 Progress 并记录待办标题

### Changed

- 主页职业状态区、创建任务、长休总结、同步页云端任务列表支持 12 职业
- 共鸣取代旧的简单队伍协同奖励，奖励更细分并可持续成长
- 任务地图背景跟随新增区域扩展，长期任务推进更有探索感
- README、AGENTS 文档同步更新为 v1.7 / persist v12
- 长休总结改为按 `ProgressLog.className` 精确统计职业进度，卷轴使用会记录技能事件日志
- 推进奖励计算抽离为纯函数，便于后续平衡调整和测试

### Fixed

- 数据迁移至 v10：自动补齐新增职业状态、共鸣图鉴、共鸣 Buff、共鸣链状态
- 数据迁移至 v11：为历史日志补齐 `type` 与 `className`
- 数据迁移至 v12：为任务补齐独立 Todo List，并为待办推进日志记录 `todoId` / `todoTitle`
- WebDAV 空本地存档不会再覆盖已有云端存档，并在替换本地/云端前增加二次确认

## v1.6 - 2026-05

新增**优先级与休息系统**，让 QuestFlow 更接近管理一支 DnD 冒险队伍的体验。

### Added

- 任务标签：重要（🔵）/ 紧急（🔴），可同时选择（🟣 +5 XP）
- 职业疲劳值系统（0~100），每次推进 +5 疲劳
- 短休（5 分钟）：所有职业恢复 30% 疲劳
- 长休（15 分钟）：所有职业疲劳归零，触发今日冒险总结
- 队伍协同奖励：连续推进不同职业时 +10 XP + 卷轴加成
- Party Status 面板展示各职业 XP 和疲劳条

### Changed

- 疲劳影响奖励：>30 → 90%，>60 → 75%，>80 → 50%

### Fixed

- 数据迁移至 v7（自动处理结构变更）

## v1.5 - 2026-04

大版本更新，核心变更：**用 DnD 职业成长体系取代了伙伴抽卡系统**。

### Added

- 5 职业体系，每个职业绑定真实工作类型
- 技能检定（d20 投骰）机制
- 卷轴 → 技能线学习/升环系统（1~9 环，2^(n-1) 指数升环）
- 法术书（Spellbook）界面
- 卷轴开启动画（ScrollReveal）

### Changed

- 所有技能/检定/卷轴名称中文化

### Removed

- 水晶、伙伴抽卡、伙伴图鉴、陪伴伙伴

### Fixed

- 数据迁移至 v5（自动处理结构变更）
